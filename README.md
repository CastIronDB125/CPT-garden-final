# GPT-garden-final

Final blended build for the Bella Vista garden tracker.

## Includes
- dark theme and typography inspired by the Claude blend
- richer live weather briefing from the GPT builds
- sunrise, sunset, daylight hours, 7-day forecast
- crop-specific hourly outdoor window
- auto/manual crop profile toggles
- destination-aware indoor vs outdoor guidance
- plant inventory, logs, feed records, DWC logs
- export/import preserving mutable plant state
- normal page scrolling and weather failure fallback

## Publish on GitHub Pages
1. Create a repo named `GPT-garden-final`
2. Upload the contents of this folder to the repo root
3. In **Settings → Pages**, deploy from `main` and `/ (root)`
4. Wait a minute, then open the site URL GitHub gives you
